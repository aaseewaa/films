# Функциональное тестирование (ручное)

Проверка пользовательских сценариев по чек-листам. Запуск: фронт `npm run dev`, бэкенд `uvicorn app.main:app --reload`, БД с данными.

## Авторизация

- [ ] Регистрация нового пользователя
- [ ] Вход с корректным паролем
- [ ] Ошибка при неверном пароле
- [ ] Выход и повторный вход
- [ ] `/api/auth/me` с токеном и без

## Каталог и карточки

- [ ] Список фильмов: сортировка, пагинация
- [ ] Карточка фильма: постер, рейтинг, создатели
- [ ] Карточка персоны: биография, фильмография, граф влияний
- [ ] Поиск по названию / имени

## Пользовательские данные

- [ ] Избранное: добавить / убрать фильм
- [ ] Оценка фильма
- [ ] История просмотров

## Прочее
