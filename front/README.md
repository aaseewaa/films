# FilmCine — фронтенд

Информационная система о кино: граф влияний режиссёров, семантический поиск, журнал статей.

## Стек

- **Vite** + **React 18** + **TypeScript**
- **Tailwind CSS** — стили
- **React Router 6** — навигация
- **TanStack Query** — работа с API
- **Zustand** — авторизация
- **react-force-graph-2d** — интерактивный граф

## Запуск

### 1. Установить зависимости

```bash
cd front
npm install
```

Это займёт несколько минут — скачивает ~300 МБ.

### 2. Запустить бэк (отдельный терминал)

```bash
cd backend
source venv/bin/activate
uvicorn app.main:app --reload
```

Бэк должен слушать `http://localhost:8000`. Vite прокидывает `/api/*` туда.

### 3. Запустить фронт

```bash
cd front
npm run dev
```

Открыть `http://localhost:5173`. Должен появиться **живой граф топ-50 режиссёров**.

## Структура

```
src/
├── api/              ← Типизированные HTTP-вызовы к бэку
├── components/
│   ├── layout/       ← Header, меню, поиск
│   └── ui/           ← Кнопки, карточки, базовые
├── pages/            ← Страницы маршрутов
├── stores/           ← Zustand для авторизации
├── hooks/            ← Кастомные React-хуки
├── lib/              ← Утилиты (cn для tailwind)
└── styles/           ← Глобальные стили + tailwind
```

## Дизайн-токены

- Фон: `bg-cream-100` (#FAF6EE)
- Текст: `text-ink-300` (тёмно-серый)
- Акцент: `text-wine-500` (винный)
- Тёплый акцент: `text-butter-400` (сливочное масло)
- Тёмный режим (только страница графа): `bg-graph-bg`

Заголовки — serif (Playfair Display), текст — sans (Inter).
