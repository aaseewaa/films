# Radial Graph — переделка главной страницы

## Что меняется

Главная страница перестаёт быть «облаком точек» и становится **звездой**:
- В центре — большой фото-аватар режиссёра (200px)
- Вокруг — ровно **4 узла** на позициях 12 / 3 / 6 / 9 часов
- Узлы — топ-4 по силе связи (`director_influence.weight`)
- Если у режиссёра меньше 4 связей — пустые места становятся серыми пунктирными заглушками
- Никакого `react-force-graph`, никакой физики
- Чистый SVG + детерминированная математика
- Анимации через CSS transitions

## Что в архиве

```
backend/
├── radial_graph_service.py    ← новый сервис
└── graph_endpoint_patch.py    ← кусок кода для вставки в api/graph.py

frontend/
├── graph.ts                    ← заменяет src/api/graph.ts
└── HomePage.tsx                ← заменяет src/pages/HomePage.tsx
```

### Шаг 2. Бэкенд — добавить эндпоинт в роутер

Открой `backend/app/api/graph.py` и добавь:

**В начале файла** (там где другие импорты сервисов):
```python
from app.services.radial_graph_service import RadialGraphService
```

**В конце файла** (после всех существующих @router.get) — содержимое из `graph_endpoint_patch.py`, БЕЗ верхних импортов и без закомментированной строки `# router = ...`. То есть просто:

```python
@router.get(
    "/director/{center_id}/radial",
    summary="Радиальная карточка: центр + топ-N значимых связей (глубина 1)",
)
async def director_radial(
    center_id: int,
    top_n: Annotated[int, Query(ge=1, le=8, description="Сколько соседей вернуть")] = 4,
    lang: Annotated[Literal["ru", "en"], Query()] = "ru",
    db: AsyncSession = Depends(get_db),
):
    service = RadialGraphService(db)
    result = await service.get_radial(center_id, top_n=top_n, lang=lang)
    if result is None:
        raise HTTPException(
            status_code=404,
            detail=f"Режиссёр {center_id} не найден или не помечен как is_director",
        )
    return result
```

Если в `graph.py` ещё НЕТ импортов `Annotated, Literal, Query` — добавь их в верхушку:
```python
from typing import Annotated, Literal
from fastapi import APIRouter, Depends, HTTPException, Query
```

Uvicorn перезагрузится сам.

### Шаг 5. Проверь в браузере

Открой `http://localhost:5173/`.

Должна быть:
- Большой фото-аватар в центре (200px)
- 4 узла вокруг на позициях 12/3/6/9 часов (140px)
- Тонкие лучи от центра к узлам
- Подписи имён под каждым аватаром
- Если связей у режиссёра меньше 4 — серые пунктирные круги вместо реальных узлов

При наведении на узел:
- Узел чуть увеличивается (на 6%)
- Подпись исчезает
- Луч-связь становится золотым
- Обводка узла становится золотой

При клике на узел:
- Узел становится новым центром
- Граф плавно перестраивается (новый запрос к API)

При двойном клике на узел — переход на карточку режиссёра `/director/:id`.

При клике на центр — переход на его карточку.

Кнопка справа сверху «↻ Случайный режиссёр» — выбирает новый центр из топ-10 любимчиков.

## Что важно знать

### Веса связей

Эндпоинт сортирует связи по `director_influence.weight` (это поле есть у тебя в схеме). Если у разных рёбер одинаковый weight — сортирует по `films_count` (числу фильмов режиссёра в БД), чтобы более «весомые» режиссёры выходили вперёд.

### Симметричные связи

Связь Хичкок→Линч и Линч→Хичкок считаются как **одна связь** через `UNION` направлений в SQL. Не будет дублей.

### Глубина строго 1

Радиальная карточка показывает только прямых соседей. Это **сознательное** решение для визуальной чистоты. Если пользователь хочет «дальше» — он кликает на узел и тот становится новым центром (виртуальное углубление по графу).

### Если связей в БД < 4

Например у Соррентино может быть всего 1 связь в твоей БД. На радиальной карточке:
- 1 реальный узел + 3 серые заглушки
- Серые заглушки — пунктирная окружность, без имени, без интерактива
- Визуально граф всё равно «полный» и красивый
- На защите можно сказать: «заглушки показывают потенциальные связи, которые могут быть добавлены через NLP-анализ биографий или экспертную разметку»

## Возможные проблемы

### 404 на новом эндпоинте
Не перезапустился uvicorn или импорт `RadialGraphService` не добавлен. Проверь импорт.

### `weight is null` в БД
Если у тебя в `director_influence.weight` пустые значения — SQL использует `COALESCE(weight, 1.0)`. Все связи будут равны по весу, фильтр пойдёт по `films_count`. Это нормально для первой версии.

### Аватар не круглый, обрезается квадратом
Проверь что в браузере открылся именно SVG. Если Cursor показывает старую закешированную HTML — `Cmd+Shift+R` для жёсткой перезагрузки.

### `transformBox` не поддерживается
Если в браузере не работает плавное масштабирование при hover — это известная проблема старых Safari. На Chrome работает.
